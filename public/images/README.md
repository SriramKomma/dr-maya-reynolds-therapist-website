## Image Optimization Tips

1. **Format**: Use WebP for best compression, with JPG fallback
2. **Size**: Optimize all images before upload (use tools like TinyPNG)
3. **Naming**: Use descriptive, lowercase names with hyphens
4. **Alt Text**: Always provide descriptive alt text for accessibility

## Color Palette Reference

Ensure images complement these colors:
- Primary: Sage green (#4a7d76)
- Secondary: Warm beige (#b89b7d)
- Accent: Soft coral (#ffa764)
- Neutral: Stone tones (#fafaf9 to #1c1917)
